What is Grow King Scripts?

Grow King Scripts (GKS) is a lua script created by @Grow King Scripts to automate the Growtopia game.

How to buy GKS script?

1. Join to the Grow King Scripts Discord server. Join now!

2. Create a purchase ticket and type the command /gks-help.

3. Drop WL or DL in the specified world.

4. Screenshot the payment and send it to the ticket you have already created.

5. If the GrowID displayed in the screenshot is the same as the GrowID displayed on the CCTV in the world, then the Ticket Admin will immediately add the balance to you. [If the payment uses Saweria, enter the information with your username + discord tag and if the payment is successful it will be in the Saweria payment log.]
